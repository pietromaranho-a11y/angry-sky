# 🛒 Meu Catálogo

Este é um projeto de catálogo digital desenvolvido para listar e gerenciar itens de forma dinâmica, integrando uma interface responsiva com um banco de dados em tempo real.

## 🚀 Funcionalidades

- Exibição de produtos/itens cadastrados.
- Conexão em tempo real com o banco de dados.
- Layout adaptável para dispositivos móveis e desktop.

## 🛠 Tecnologias Utilizadas

- **HTML5** & **CSS3**: Estrutura e estilização.
- **JavaScript (ES6+)**: Lógica de renderização e consumo de API.
- **Supabase**: Backend-as-a-Service (Banco de dados e Autenticação).
